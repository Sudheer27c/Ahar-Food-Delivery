# 🍽️ AHAR - Food Delivery Web Application

## 📌 Project Overview

AHAR is a Java Full Stack Food Delivery Web Application developed using Java, Servlets, JSP, JDBC, MySQL, HTML, CSS, and JavaScript. The application allows customers to browse restaurants, order food online, manage their cart, download invoices, and manage their profile. It also provides an Admin Dashboard to manage restaurants, menus, orders, users, coupons, and reports.
